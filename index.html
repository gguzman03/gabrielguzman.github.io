Boss Rush

Boss Rush was my first time actually making a game, made while still pursuing my CS Degree. The game has an intentionally simple but coherent design, being a simple platformer with a crucial item to attain and win the game. The game has a fully functional win/loss condition system, as well as a competent settings menu.



This project was made using the Flutter framework and Dart programming language, with the Tiled program for platforming functionality. PixilArt was used for the sprites.

Boss Chaos!

This is the direct sequel to Boss Rush, adding enhanced gameplay and mechanics such as an invincible dash and a melee attack. Simplicity is further embraced with a simple premise behind the game, animated with Adobe Animate.



This time, Unity was used for programming all the game functionality, from gameplay to UI. Later on, I added a bonus mode called Boss Chaos: Blood Rush Mode, which uses an Arduino Heartbeat sensor to enable a much higher pace of gameplay.

Disgruntled Ducks

I was part of a team that would make an interactive AR game. As a game, Disgruntled Ducks borrowed key elements from Angry Birds, featuring a slingshot to launch marbles and destroy obstacles, the game’s objective being freeing trapped ducks. I was mainly in charge of game mechanic duties. The user would roll a dice that would be scanned by the user’s camera, given one of four elements (Fire, Water, Earth, and Air) that would determine what marble would be used. Each elemental marble had its own properties. Gameplay was once again made in Unity, and Vuforia was our AR software of choice.
